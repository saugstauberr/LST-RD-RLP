Zurück zum Umwandler

Version 1.0.0

Veröffentlicht: 06.04.2025
  • Leitstellen-Einsatzmeldungen können von Zahlen in Meldung umgewandelt werden
  • Einen "Umwandel"-Button gibt es nicht mehr, Text wird direkt nach eingeben umgewandelt
  • Zahlen können einfach hintereinander geschrieben werden (ein Code ist immer 3 Zahlen lang, danach wird automatisch ein Leerzeichen eingefügt)
