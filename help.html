Zweck der Seite

Die Leitstelle versendet bei jedem Einsatz eine Rückmeldeindikation (kurz: RMI). Diese wird als 3-stellige Zahl (z.B. 801) verschickt, jedoch nicht immer als Klartext angezeigt.
Falls Sie die Bedeutung der RMI nicht kennen, können Sie diese hier eingeben und die Bedeutung wird Ihnen angezeigt.

Verwendung

Geben Sie auf der Startseite einen 3-stelligen Code ein. Dieser wird dann entschlüsselt.
Sie können auch mehrere Codes gleichzeitig eingeben, indem Sie diese einfach hinteeinander eintippen. Sie werden automatisch voneinander getrennt.
Beispiel: 801 802 803
Die Bedeutung wird Ihnen dann in einer Liste angezeigt.
